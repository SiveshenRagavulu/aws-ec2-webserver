# EC2 Web Server Deployment

## Description
Hands‑on AWS project that demonstrates launching an Amazon EC2 instance in the Cape Town region (af-south-1). The goal was to create and connect to a virtual server in AWS. The process was documented with screenshots to show each step clearly.
## Objective
To learn how to launch and configure an EC2 instance in AWS.

## Architecture
- EC2 instance (t3.micro, Windows)
- VPC with public subnet
- Security Groups for HTTP/SSH
- IAM role for S3 access
- Elastic IP

## Implementation
Launched an EC2 instance in Cape Town region (af-south-1)
- Created a key pair for secure access
- Set up a security group to allow connection
- Connected to the instance using the key
- Added screenshots to document the process
- 
## Outcome
- EC2 instance successfully launched and running
- Able to connect to the instance with the key
- Deployment steps completed and recorded
- Screenshots uploaded to GitHub as proof of success
