# EC2 Web Server Deployment

## Description
Hands‑on AWS project demonstrating the launch and configuration of an EC2 instance in the Africa (Cape Town) Region. Includes setup of VPC, Security Groups, IAM roles, and Elastic IP. Automated Apache installation via User Data script to host a sample web application. Documentation covers architecture, implementation steps, and outcomes.

## Objective
To learn how to launch and configure an EC2 instance in AWS.

## Architecture
- EC2 instance (t2.micro, Amazon Linux 2)
- VPC with public subnet
- Security Groups for HTTP/SSH
- IAM role for S3 access
- Elastic IP
